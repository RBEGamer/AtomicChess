Parameter size

The parameter size sets the whole width of the game (including the board). Depending in which mode you are working, the result may be different:
  1. In case of board, set the size of the board.
  2. in case of print, set the size of the column used for printing the game.
  3. In case of view, it sets the size of the board and the column to print the moves.
  4. In case of edit, it sets the size of the board, the comment field (and others), and the size of the column to print the moves.
