Parameter boardSize

The following boards (and moves) set the parameter boardSize:
  1. Uses the default of Chessground, not the div: 320px
  2. Uses the parameter width: 450px
  3. Explicit to set the board width: 600px
  4. Explicit to set the board width big: 450px, coordsInner: false
  5. Explicit to set the board width small: 200px, width: 400px (for the moves)
  6. Explicit to set the board width small: 200px (board and moves)
  7. Explicit to set the board width: 300px, moves: 400px
