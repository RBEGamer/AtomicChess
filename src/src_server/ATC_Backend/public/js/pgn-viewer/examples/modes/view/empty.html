Minimal configuration

This example shows the minimal configuration when using the distributed version of PgnViewerJS. There are only three files needed:
  • dist/css/pgnvjs.css: The concatenated css files of the directory CSS.
  • font-awesome.min.css: Taken from the internet, may be part later of the concatenated version ...
  • dist/js/pgnv.js: The concatenated js files of the directory js (and all the other plugins used).
