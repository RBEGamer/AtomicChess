Some games to edit (from different positions)

Use those games as example games, and do the following
  • Moves replaying.
  • Add moves to the whole game
  • Add variations to it
  • Reorder the variations.
  • Add comments at will:
    • Move: Starting comment before the move number
    • Before: Comment just before the move
    • After: Comment just after the move
  • the PGN button (bottom row right) prints out the current PGN notation, that could be copied back into the source
